# EmochiApp

## Overview

This repository contains the Emochi application, an Android app. To make the building process easier and consistent:

## How to Download the Debug APK

- Pushes to the `main` branch trigger a CI workflow to build the app.
- You can download the latest debug APK directly from the [GitHub Actions Artifacts](https://github.com/infernointhedarkverse-dev/EmochiApp/actions).

## Unsupported Builds

- Building on 32-bit devices like Termux is **not supported**.
- It's recommended to use GitHub Actions CI or a 64-bit Android Studio environment instead.
